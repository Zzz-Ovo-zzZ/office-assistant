## 语言偏好

永远使用中文回复用户。所有对话、解释、代码注释都应使用中文。

## 项目定位

办公效率 + 生活规划工具集，涵盖：

| 模块 | 说明 |
|------|------|
| 办公自动化 | 日常工作流程自动化脚本 |
| 生活规划 | 婚嫁择日、事件筹备、预算管理、行程安排 |
| 知识管理 | 备忘录、决策记录、经验沉淀 |

## 目录结构

```
办公助手/
├── scripts/        # 可执行脚本
├── output/         # 输出文件（计划、文档、报告）
├── temp/           # 临时文件

```
